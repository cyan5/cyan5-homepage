# cyan's Homepage
cyan5のホームページ

## 公開ページ
https://cyan5.github.io/cyan5-homepage/

## 開発環境
Ubuntu on WSL2<br>
React + TypeScript + Vite<br>
CSS-in-JS: Linaria

## 実装について
自作WebApp 2作品目。1作品目のReact World WeatherではCSS-in-JSにstyled componentsを採用していたが、本作品ではstyled componentsと同じ記述ができ、ゼロランタイムで高速なLinariaを採用している。
